# Exemplos de relatórios

Três exemplos de saída do GH Dev Analyzer, mostrando diferentes perfis de desenvolvedores.

| Arquivo | Tipo de perfil | Descrição |
|---------|----------------|-----------|
| [exemplo-01-iniciante.html](exemplo-01-iniciante.html) | Iniciante | Poucos repositórios, 1–2 linguagens, poucos seguidores |
| [exemplo-02-intermediario.html](exemplo-02-intermediario.html) | Intermediário | Vários projetos, linguagens variadas, topics, boa atividade |
| [exemplo-03-avancado.html](exemplo-03-avancado.html) | Avançado | Muitos repos, alta visibilidade, OSS, diversos topics |

Abra os arquivos `.html` no navegador para ver como fica o relatório gerado pela aplicação.
